# Ethics Checklist

Complete this before your final submission.

## Data Collection
- [ ] All data collected from public sources only
- [ ] No personally identifiable information (PII) stored
- [ ] Sources documented in README

## Model Use
- [ ] API keys not committed to GitHub
- [ ] Model outputs reviewed before publication
- [ ] Temperature and parameters documented

## Representation
- [ ] Agents are clearly labeled as simulations, not real people
- [ ] The application includes a disclaimer about the simulation nature
- [ ] Results are not presented as ground truth

## GDPR
- [ ] No user data collected by the application
- [ ] No cookies or tracking
